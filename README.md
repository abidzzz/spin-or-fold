#  SLOT - Poker Slot Machine

A roguelike poker-slot hybrid game built with Godot Engine

> **⚠️ NOTE:** This project was built in ~6 hours for a deadline. It has bugs and rough edges. The core concept works, but expect issues. Future updates may fix things.

---

## What's Supposed to Happen

- Select your bet hand (Pair → Four of a Kind)
- Press S to spin, SPACE to stop reels
- Win if your hand beats your bet!
- 6 rounds to escape the casino
- Choose upgrades after each round

---

## Known Issues

- LOCK upgrade is temperamental
- Boss round can be inconsistent
- Some visual glitches
- Various other bugs from the 6-hour crunch

---

## Controls

| Key | Action |
|-----|--------|
| `S` | Spin the reels |
| `SPACE` | Stop next reel |
| `ESC` | Skip upgrade selection |

---

## Upgrades

| Upgrade | Effect |
|---------|--------|
| WEIGHT | Reel favors a suit |
| LOCK | Reel holds its card |
| SLOW | Reel spins slower |
| WILD | Wild cards appear |
| DOUBLE | Card counts twice |

---

## 🚀 How to Run

1. Download Godot Engine 4.6+
2. Clone this repository
3. Open project in Godot
4. Press F5 to run

## The Story

This was built in a single 6-hour session (5 AM to 10 AM) for a deadline. 

**Things that went wrong:**
- A lot of bugs discovered at the last minute
- LOCK upgrade refused to work properly
- Visuals looked rough
- Sleep deprivation didn't help

**What went right:**
- Core slot machine works
- Hand evaluation works
- Upgrade system works (mostly)
- It's a complete game loop

I'm uploading it anyway because:
1. It's better to ship than to ghost
2. Future me might fix it
3. Someone might find it interesting
